## Foundational

### Deep Learning (Goodfellow, Bengio, Courville)
| Field | Value |
|-------|-------|
| **URL** | https://www.deeplearningbook.org/ |
| **Why useful** | Definitive textbook on deep learning theory — covers linear algebra, probability, neural networks, CNNs, RNNs, autoencoders, generative models with mathematical rigor |
| **Difficulty** | Advanced |
| **Prerequisites** | Calculus, linear algebra, probability basics |
| **Best time to use** | Phases 1–3 (as reference); read chapters alongside phase content |
| **Estimated time** | 40+ hours (reference, not cover-to-cover) |
| **Free/Paid** | Free (HTML) |
| **Actively maintained** | No (last updated 2016, still foundational) |

### Mathematics for Machine Learning (Deisenroth, Faisal, Ong)
| Field | Value |
|-------|-------|
| **URL** | https://mml-book.github.io/ |
| **Why useful** | Covers the exact math you need: linear algebra, analytic geometry, matrix decompositions, probability, optimization — with ML context throughout |
| **Difficulty** | Intermediate |
| **Prerequisites** | High-school math |
| **Best time to use** | Phase 1 (primary math reference) |
| **Estimated time** | 25–35 hours |
| **Free/Paid** | Free (PDF) |
| **Actively maintained** | No (last updated 2020, content is stable) |

### Understanding Deep Learning (Simon Prince)
| Field | Value |
|-------|-------|
| **URL** | https://udlbook.github.io/udlbook/ |
| **Why useful** | Most modern free DL textbook — covers transformers, diffusion models, structured prediction with excellent visualizations and Python notebooks |
| **Difficulty** | Intermediate |
| **Prerequisites** | Python, calculus basics |
| **Best time to use** | Phases 2–4 (covers ML → DL → transformers) |
| **Estimated time** | 30–40 hours |
| **Free/Paid** | Free (PDF) |
| **Actively maintained** | Yes (updated 2024) |

### The Little Book of Deep Learning (François Fleuret)
| Field | Value |
|-------|-------|
| **URL** | https://fleuret.org/public/lbdl.pdf |
| **Why useful** | Concise 150-page reference covering tensors, training, architectures, attention — perfect for review before interviews |
| **Difficulty** | Intermediate |
| **Prerequisites** | Basic ML knowledge |
| **Best time to use** | Phase 3 review, or pre-interview revision |
| **Estimated time** | 5–8 hours |
| **Free/Paid** | Free (PDF) |
| **Actively maintained** | Yes (updated 2023) |

## Applied / Practitioner

### Hands-On Machine Learning with Scikit-Learn, Keras & TensorFlow (Aurélien Géron)
| Field | Value |
|-------|-------|
| **URL** | https://www.oreilly.com/library/view/hands-on-machine-learning/9781492032632/ |
| **Why useful** | Best hands-on ML book — covers scikit-learn, data pipelines, neural networks, CNNs, RNNs, attention, deployment with working code throughout |
| **Difficulty** | Intermediate |
| **Prerequisites** | Python basics |
| **Best time to use** | Phases 2–3 |
| **Estimated time** | 30–40 hours |
| **Free/Paid** | Paid ($65) / O'Reilly subscription |
| **Actively maintained** | Yes (3rd edition 2023) |

### Natural Language Processing with Transformers (Lewis Tunstall, Leandro von Werra, Thomas Wolf)
| Field | Value |
|-------|-------|
| **URL** | https://www.oreilly.com/library/view/natural-language-processing/9781098136789/ |
| **Why useful** | Written by HuggingFace core team — the definitive practical guide to transformers: tokenization, text classification, NER, translation, summarization, RAG, fine-tuning |
| **Difficulty** | Intermediate |
| **Prerequisites** | Python, basic ML |
| **Best time to use** | Phases 4–5 |
| **Estimated time** | 20–25 hours |
| **Free/Paid** | Paid ($60) / O'Reilly subscription; notebooks free on GitHub |
| **Actively maintained** | Yes (2nd edition in progress) |

### Designing Machine Learning Systems (Chip Huyen)
| Field | Value |
|-------|-------|
| **URL** | https://www.oreilly.com/library/view/designing-machine-learning/9781098107956/ |
| **Why useful** | Production ML systems — feature engineering, training pipelines, model evaluation, deployment, monitoring, data quality; written by someone who shipped systems at scale |
| **Difficulty** | Advanced |
| **Prerequisites** | ML fundamentals |
| **Best time to use** | Phase 9 (Production AI Engineering) |
| **Estimated time** | 15–20 hours |
| **Free/Paid** | Paid ($60) / O'Reilly subscription |
| **Actively maintained** | Yes (2022, remains current) |

### Building LLMs for Production (Towards AI team)
| Field | Value |

| **Best time to use** | Phases 5–9 |
| **Estimated time** | 15–20 hours |
